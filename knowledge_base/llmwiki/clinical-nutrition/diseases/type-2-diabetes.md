# 2型糖尿病营养管理总原则

## 适用人群

本页适用于成人 2 型糖尿病、糖尿病前期、超重或肥胖合并血糖异常的人群。儿童、妊娠期糖尿病、1 型糖尿病、进食障碍、严重肾病或反复低血糖用户，需要转入专门方案，不能直接套用普通成人建议。

## 核心目标

糖尿病营养管理不是简单“少吃糖”，而是用可长期执行的饮食结构帮助用户稳定血糖波动、改善体重和胰岛素抵抗、降低心血管和肾脏并发症风险。

Agent 回答时要同时关注四个目标：

- 餐后血糖更平稳。
- 总能量和体重管理更可持续。
- 蛋白质、脂肪、膳食纤维和微量营养素不过度失衡。
- 用户能在真实生活里执行，而不是只得到一句“不能吃”。

## 推荐行动

1. 先判断用户是否已经确诊糖尿病、是否在用降糖药或胰岛素、是否有低血糖史。
2. 询问或利用记忆确认用户的身高、体重、体力活动、血糖控制目标、肾功能和医生限制。
3. 对每一餐先看“碳水种类和份量”，再看蛋白质、蔬菜、油脂和烹调方式。
4. 鼓励规律进餐，避免一餐大量精制主食或含糖饮料。
5. 给替换方案时优先使用“减少、替换、搭配、分量调整”，避免制造恐惧。

## 餐盘结构

如果用户没有特殊禁忌，可以用“半盘非淀粉蔬菜、四分之一优质蛋白、四分之一全谷物或杂豆薯类”的思路解释一餐结构。这个结构不是硬性比例，而是帮助用户理解控糖餐的形状。

常见可选项：

- 非淀粉蔬菜：绿叶菜、番茄、黄瓜、西兰花、菌菇。
- 优质蛋白：鱼、禽、蛋、奶、豆制品、瘦肉。
- 主食：燕麦、全麦制品、糙米、杂豆、薯类；减少白粥、甜面包、含糖点心。
- 脂肪：优先植物油、坚果少量；减少反复油炸和高饱和脂肪加工食品。

## Agent 回答约束

- 不要把糖尿病饮食说成“完全不能吃主食”。
- 不要用单一食物判断整餐是否健康，要看总量、搭配和血糖反应。
- 如果用户提到低血糖、胰岛素、磺脲类药物、肾功能不全、妊娠、足部感染或酮症相关症状，应提示就医或遵医嘱调整。
- 如果用户要求制定严格减重、极低碳水或断食方案，应先提醒存在低血糖和营养不足风险。

## 常见误区

- “不吃主食就能控糖”：短期可能降低血糖，但长期容易影响执行、诱发低血糖或导致营养结构失衡。
- “无糖食品可以随便吃”：仍要看淀粉、脂肪和总能量。
- “粗粮越多越好”：粗粮也含碳水，胃肠耐受差或肾病用户需要更谨慎。
- “水果都不能吃”：应看种类、份量、进食时间和个体血糖反应。

## 对话示例规则

用户说：“我早餐吃两个鸡蛋、一杯牛奶、两片面包，糖尿病能吃吗？”

回答应先判断：鸡蛋和无糖奶通常可作为蛋白质来源，真正需要重点评估的是面包种类、片数、是否搭配蔬菜、餐后血糖反应。建议把白面包优先替换为全麦或杂粮主食，并观察餐后 2 小时血糖。
